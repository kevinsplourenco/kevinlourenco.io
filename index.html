Discord Clone

Um clone completo de ponta a ponta (end-to-end) e em tempo real do Discord, incluindo servidores, canais, chamadas de vídeo, chamadas de áudio, edição e exclusão de mensagens, assim como funções de membros.

  • Mensagens em tempo real usando Socket.io;
  • Envio de anexos como mensagens usando UploadThing;
  • Criação de canais de chamadas de texto, áudio e vídeo;
  • Total responsividade e UI para dispositivos móveis.

Instagram Clone

Clone do Instagram de ponta a ponta, utilizando React Native, Autenticação com Firebase, Spring Boot e API REST.

  • Integrando o Front-End com REST API e Firebase.
